# Law 65: refer.design.md - Design Standards (Agnostic)

Canonical UX and design standards live in REFER.OS. Start with:

- `REFER.OS/refer.structure.md`
- `REFER.OS/refer.designlab.md`
- `REFER.OS/refer.seamless-ui.md` (Flat Seamless UI pattern for borderless, section-surface page builds)

App-specific design deviations or extensions belong in app state law under `refer.app/` (for example, `refer.app/refer.telechurch.design.md`).
