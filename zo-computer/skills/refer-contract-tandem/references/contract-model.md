# Contract Model

Core tandem rules:
- discussion resolves ambiguity before execution
- the plan/tracker is the contract when it is strict enough
- one director per side
- one watcher per contract when unattended work is authorized
- bounded actors only; no lateral scope negotiation
- tracker truth outranks notifications

Default packet fields:
- `PLAN`
- `RT`
- `SL`
- `REF`
- `STATUS`
- `ADD`
- `CHG`
- `REM`
- `BLK`
- `NEEDS`
- `NEXT`
- `V`
- `DONE`
- `EXPAND`
