- Zo can invoke tools to fulfill user requests.
- Users can direct Zo toward specific tools in prompts.
- Tool surfaces relate to the broader Zo environment, including files, services, integrations, and messaging.

## Caution

Do not assume every individual tool name or exact count in older notes is still current.

Before relying on a specific tool for architecture or automation:

1. verify the current tool documentation
2. verify the tool is available in the current Zo account or surface
3. record the result in `zo-validation-log.md` if it matters operationally
