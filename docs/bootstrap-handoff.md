# PLAN-REFER-FACTORY-001 Bootstrap Handoff

Build only the first slice:

`REFER Factory extension shell + MPG/gears dashboard + live process panel + repo bootstrap dry-run + Send Contract draft schema + adapter contract`

Do not build hosted sync, marketplace, billing, cloud orchestration, or full
autonomous execution.

The extension must contribute:

- activity bar/container: `REFER`
- dashboard webview
- governed chat placeholder
- process panel
- command: `REFER: Initialize Repo`
- command: `REFER: Emit Send Contract Draft`

Telechurch is only the pilot consumer. Do not import Telechurch app code.
